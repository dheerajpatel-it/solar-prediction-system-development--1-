🌞 Solar Energy Prediction System

A modern web-based application to estimate and analyze solar energy generation based on location, building characteristics, and environmental factors.

🚀 Features
📍 Location-Based Analysis
Enter geographic details to estimate solar potential.
🏢 Building Configuration
Customize roof size, tilt, and orientation.
⚡ Energy Consumption Profiling
Analyze energy usage patterns.
🌦️ Environmental Factors
Consider sunlight, weather, and surroundings.
📊 Interactive Dashboard
Visualize predicted solar output and insights.
📄 Project Report Generation
Generate a structured report for analysis.
🛠️ Tech Stack
Frontend: React + TypeScript
Build Tool: Vite
Styling: CSS
Architecture: Component-based modular design

⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/your-username/solar-prediction-system.git
Navigate to the project directory:
cd solar-prediction-system
Install dependencies:
npm install
Run the development server:
npm run dev
Open in browser:
http://localhost:5173
📈 How It Works
User enters location details
Defines building parameters
Inputs energy consumption
System evaluates environmental factors
Generates solar energy prediction & report
